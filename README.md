## Setup  
1. Clone repo:  
   ```bash  
   git clone https://github.com/GauravKarakoti/socialsphere  
   cd socialsphere
   ```
2. Install dependencies:
  ```bash
  yarn install
  ```
3. Configure environment:
   ```bash
   cp .env.example .env
   ```
4. Run Locally
   ```bash
   yarn dev
   ```
